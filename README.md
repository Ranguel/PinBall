# PinBall

Physics-based pinball prototype made in Unity (1:1 scale, real-world dimensions).

## Video demo

https://github.com/user-attachments/assets/a3d7b1f7-745a-4952-8740-7dc2f8377d87

## Version

**Version:** 0.0.1-pre

## Requirements

- Unity 6.3 LTS

## Assets

This demo uses FBX 3D models with simple textures.

## Basic controls

| Action                         | Input |
|--------------------------------|-------|
| Left flipper                   | ←     |
| Right flipper                  | →     |
| Plunger (hold to charge)       | ↓     |
